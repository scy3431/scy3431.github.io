Estimating True CAPM Betas for the Magnificent 7 [GitHub]

Python · Statsmodels · SciPy · yfinance  ·  Dec. 2025 – Jan. 2026

Estimated CAPM betas for AAPL, MSFT, GOOGL, AMZN, META, NVDA, and TSLA via OLS regression on excess returns (2015–2025), correctly subtracting the 3-month T-bill risk-free rate.
